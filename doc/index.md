# cwipc Documentation

Welcome to the **cwipc** documentation.  This repository contains libraries and tools
for working with dynamic point clouds.  The documentation is organised in a
ReadTheDocs style with high‑level sections covering installation, command‑line
utilities, hardware setup, programming interfaces and development topics.

The table of contents below mirrors the section structure that will appear on
ReadTheDocs.  Click through for details.
